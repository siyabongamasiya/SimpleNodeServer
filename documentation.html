Media API Documentation

This API allows you to manage a collection of Movies, Series, and Songs stored in a JSON file.

Movies

  • GET /Movies - Get all movies or query with ?name=MovieName to get one.
  • POST /Movies - Add a new movie. Send JSON: 
    {
  "name": "Inception",
  "releaseDate": "2010",
  "starRating": 9,
  "rating": 13
}
  • PUT /Movies - Update an existing movie (by name).
  • DELETE /Movies/:name - Delete a movie by name.

Series

  • GET /Series - Get all series or query with ?name=SeriesName.
  • POST /Series - Add a new series. JSON format: 
    {
  "name": "Breaking Bad",
  "season": "5",
  "starRating": 10,
  "rating": 18
}
  • PUT /Series - Update an existing series (by name).
  • DELETE /Series/:name - Delete a series by name.

Songs

  • GET /Songs - Get all songs or query with ?name=SongName.
  • POST /Songs - Add a new song. JSON format: 
    {
  "name": "Water",
  "artist": "Tyla",
  "genre": "Amapiano",
  "noOfStreams": 3000000
}
  • PUT /Songs - Update an existing song (by name).
  • DELETE /Songs/:name - Delete a song by name.
